The data_analysis folder contains the quantitative and qualitative data collected in the experiment as well as the scripts used to analyse it.

The raw quantitative data is contained in the folders hrtf_data and panning_data.

The raw qualitative data is contained in the FINAL_qualitative_Data.tsv file.

The quantitative results presented in the report have been achieved using the matlab scripts device_computeanalysis.m and device_computeanalysis_StatTest.m

The qualitative results have been achieved using the matlab script questionnaire_computeanalysis.m

The figures included in the report are contained in the figures folder.
